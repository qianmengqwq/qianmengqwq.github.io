butterfly重装笔记

发表于2023-10-01|更新于2023-10-02
|阅读量:

背景

总体来说，本博客是基于 https://blog.anheyu.com/posts/sdxhu.html 的教程配置的。

若你也想配置一个类似的 blog，可以参照一下上述教程

但是就目前(2023/10/1)来说很多地方都已经无法完全复刻，我甚至 npm i 不到很多依赖，有的 api 也废弃了，所以魔改还是自己研究吧

不要无脑 cv 代码，认真读读配置文档，我相信大伙最后都能成功部署的^^

~最好有一定 nodejs、前端三件套基础，至少能看懂一些 npm 的安装方式~~

写在前面

​ 最近心血来潮，突然想维护一下一年前搭建过的 hexo 博客

​ 但是！我已经忘记了之前很多配置过的内容了，也不怎么能辨认的出来当初魔改的地方，再加上之前纯小白时期犯下的很多错误，这个项目对目前的我来说，宛如一个黑盒…

所以，我决定重装部分配置，并记录下整个过程，以供日后参考。现在才明白大佬们记录安装过程的高明之处 qwq

全局吸底 aplayer

插件方式：https://github.com/MoePlayer/hexo-tag-aplayer/blob/master/docs/README-zh_cn.md(已支持metingjs)
hexo-tag-aplayer(拿不到 vip 歌曲，issues 上也有提及，有需求的话自己找 api 重写一下吧)

参考：https://butterfly.js.org/posts/507c070f/#%E6%8F%92%E5%85%A5-Aplayer-html

非常重要的一点，如果想要 aplayer 音乐插件在切换的时候继续播放，就一定把 pjax 设置为 true

pjax:
enable: true
